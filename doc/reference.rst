Reference
=========
To use the functions provided by symbolic-dynamics,
simply import the package ``symbolic_dynamics``:

>>> import symbolic_dynamics as sd

All the functions listed in the reference are imported into
``symbolic_dynamics``. For example, to refer to the function
:func:`symbolic_dynamics.sofic.dot`, one may simply write
``sd.dot``.
